## 技术栈

`Vue3.x` + `Vite` + `TypeScript` + `Uni-App` + `Less`

## 下载地址

- [最新版本](https://github.com/biaov/Tune/releases)

## 特性

- 本地音乐扫描
- 音乐排序
- 音乐播放控制
- 音乐进度条
- 音乐上一首下一首切换
- 音乐搜索
